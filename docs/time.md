## time Module
time is a builtin Module.

```ruby
import time
```

### epoch
Returns the number of seconds since the Epoch, 1970-01-01 00:00:00 +0000 (UTC).

```ruby
time.epoch() -> Number
```

### sleep
Sleep for [t] milliseconds.

```ruby
time.sleep(t:num) -> Number
```

### clock
Returns the number of clocks passed divied by CLOCKS_PER_SEC.

```ruby
time.clock() -> Number
```

### nano
Returns the number of nano seconds.

```ruby
time.nano() -> Number
```
